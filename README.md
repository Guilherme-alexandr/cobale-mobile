## 📱 CobAle Mobile

O CobAle Mobile é um aplicativo que estou desenvolvendo em Flutter com foco em negociação de débitos.
A ideia do projeto é permitir que o cliente visualize suas pendências, simule acordos e confirme negociações de forma simples e organizada.

## 🎯 Objetivo do Projeto

Criar um app mobile com fluxo completo de:

- Login
- Visualização de débitos
- Detalhamento de parcelas
- Simulação de negociação (à vista ou parcelado)
- Confirmação de acordo

O projeto está sendo desenvolvido com foco em organização de código e separação de responsabilidades, pensando já em integração futura com API real.

## 🏗️ Como organizei o projeto

Estou utilizando uma estrutura por funcionalidades (feature-based), separando:

- Models (estrutura dos dados)
- Controllers (regras de negócio)
- Pages (telas)
- Widgets (componentes reutilizáveis)

## 📲 Fluxo Atual

Splash
→ Login
→ Home
→ Lista de Débitos
→ Detalhe do Débito
→ Tela de Negociação

* Na negociação é possível:

- Escolher pagamento à vista (20% desconto simulado)
- Escolher parcelado (10% desconto simulado)
- Definir quantidade de parcelas
- Simular valores automaticamente
